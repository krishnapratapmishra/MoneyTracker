║              UNIVERSE MAGNET — How to Start                      ║
╚══════════════════════════════════════════════════════════════════╝

FIRST TIME (or new computer):
──────────────────────────────
  1. Make sure Python is installed on your computer.
     If not, download it FREE from: https://www.python.org/downloads/
       → Click the big yellow button
       → Run the installer
       → ✅ CHECK "Add Python to PATH"  ← very important!
       → Click "Install Now"

  2. Double-click   launch.py
     (it will set everything up automatically — takes ~1 minute)

  3. Your browser will open to http://myuniversemagnet:5000
     That's it! You're in.


EVERY TIME AFTER THAT:
───────────────────────
  Just double-click   launch.py
  The app opens in about 3 seconds.


IMPORTANT:
───────────
  • Keep the black launcher window OPEN while using the app.
    Closing it will stop the app.

  • Only ONE computer should run the app at a time
    (if the app lives on a shared/network drive).

  • If the browser shows "This site can't be reached",
    wait 5 seconds and refresh — the app is still starting.


NEED HELP?
───────────
  If something goes wrong, open launcher.log (same folder as launch.py)
  and send it to your administrator.
